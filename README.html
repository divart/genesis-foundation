genesis-foundation

A starting point for WordPress projects using Genesis and Foundation 6

Feel free to use this for whatever you’d like, but this is offered as is and without any support (for now).

Getting Started

  1. In the terminal, navigate to the theme's root directory.
  2. Run npm install to pull down the development dependencies
  3. Run bower install to pull down the public dependencies
  4. Run gulp to intiate the build script, and do magic
  5. Run gulp watch to make life easier
  6. Gets bizzy ma dude
